# QuizMaster

A simple and interactive **Python-based Quiz Application** that allows users to test their knowledge through multiple-choice questions. The project is beginner-friendly and demonstrates the use of Python fundamentals such as functions, loops, conditions, lists, dictionaries, and score calculation.

## ✨ Features

* Multiple-choice quiz questions
* Instant answer checking
* Final score calculation
* User-friendly interface
* Easy to add or modify questions
* Runs in the terminal/command prompt

## 🛠️ Technologies Used

* **Language:** Python 3
* VS Code
* Git and GitHub

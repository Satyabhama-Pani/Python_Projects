# 🎯Number_Guess_Game
A simpe Python-based Number Guessing Game where the player tries to guess a randomly generated number within a limited number of attempts.
## 📌Features
- Random number Generation
- Limited attempts system
- Hints for higher/lower guesses
- Score tracking
## 🛠️Technologies Used
- Python 
- Random Module
## 🚀How to Run
1. Clone the repository
    - git clone https://github.com/Satyabhama-Pani/Number_Guess_Game.git
2. Navigate to the project folder
    - cd Number_Guess_Game
3. Run the program
    - python guess.py

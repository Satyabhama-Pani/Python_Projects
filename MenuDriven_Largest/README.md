# 📃Menu Driven: Largest Number
It's a menu driven program using python, which provide user an option to choose largest between two, three and four numbers all in one place.
## 📌Features
- Interactive Menu Interface
- Find Largest Number(Variant)
- Multiple Attempts
- Loop-Based Navigation
- Beginner- Friendly Design
## 🚀How to Run
1. Clone the repository
    - git clone https://github.com/Satyabhama-Pani/Menu_Driven_LargestNo.git
2. Navigate to the project folder
    - cd Menu_Driven_LargestNo
3. Run the program
    - python largest_no.py
